# v0.6.3 Store Copy

## App Store Connect: What's New
Version 0.6.3 adds native diff tools, clearer toolbar help, and stronger iPhone/iPad workflows.

- Compare the current file with disk or compare two open tabs in the new native diff view
- New Toolbar Help explains every toolbar symbol and adapts to iPhone, iPad, and Mac
- Improved iPhone/iPad toolbar access, including a pinned Help icon on iPad
- More reliable Markdown PDF export, including complete paginated output and flexible one-page exports
- Better iPhone project-sidebar file/folder actions and new-file prompts
- Improved Markdown detection/highlighting, `.bak` and dotfile handling, themes, and support-price messaging

## TestFlight: What to Test
Please focus on diff workflows, toolbar discoverability, Markdown export, and iPhone/iPad project actions in this build.

- Compare a modified file against disk
- Compare two open tabs in the native diff view
- Open Toolbar Help from the toolbar and from the macOS Help menu
- Verify Toolbar Help cards do not clip on iPhone or iPad
- Export Markdown as paginated PDF and one-page PDF
- Use iPhone project-sidebar Open File, Open Folder, and New File actions
- Open `.bak` files and dotfiles such as `.zshrc`
