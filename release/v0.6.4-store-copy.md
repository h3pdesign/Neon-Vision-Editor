# v0.6.4 Store Copy

## App Store Connect: What's New
Version 0.6.4 improves iPad and iPhone workflows with JSON tools, toolbar customization, command menus, and hardware-keyboard fixes.

- Format JSON documents or combine JSON into one line from the Tools menu
- Customize iPhone and iPad toolbar groups in Settings
- New iPad command menus for File, Find, Tools, Help, Settings, Toolbar Help, and Welcome Tour
- Markdown files now appear as editable documents in iOS Files/share/open-in flows
- Fixed iPad hardware-keyboard search focus and Select All behavior
- Added release QA coverage for Markdown PDF export, compact layouts, and mobile toolbar/sidebar parity

## Promotional Text
Neon Vision Editor 0.6.4 adds JSON formatting tools, iPad command menus, toolbar customization, Markdown share/open-in support, and iPad hardware-keyboard fixes.

## TestFlight: What to Test
Please focus on JSON tools, iPad command menus, toolbar customization, Markdown share/open-in, and hardware-keyboard editing in this build.

- Open a JSON file and use Format JSON
- Use Combine JSON Lines and confirm the document becomes one line
- On iPhone and iPad, change toolbar group visibility in Settings
- On iPad, open File, Find, Tools, Help, Settings, Toolbar Help, and Welcome Tour from command menus
- Share or open a `.md` file from Files and confirm Neon Vision Editor appears as an editor
- Type in Find with an external keyboard and confirm focus stays in the search field
- Press Cmd+A in the editor with a hardware keyboard and confirm text selection works
